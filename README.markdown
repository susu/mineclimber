MineClimber
===========

 * basic platform game only for learning purposes (SDL2, 2D graphics)

Dependencies
============

 * CMake
 * clang 3.3 or above (or GCC 4.8 or above)
 * SDL2, SDL2_image

License
=======
 * See LICENSE (GPLv3)

Build
=====

 * ```mkdir build && cd build```
 * ```cmake ..```
 * ```make```
 * ```./mineclimber```

For i3-wm users
===============

 * Useful to add ```~/.i3/config```:

```
for_window [title="MineClimber"] floating enable
```
